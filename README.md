# NFT Explorer

Another NFT Explorer.

## To configure and run locally

1. Install the dependencies:
```
yarn
```

2. Setup `.env` file
```
ALCHEMY_APY_KEY=<alchemy_api_key>
```

3. Run the tests (optional)
```
yarn test
```

4. Run the project
```
yarn dev
```

5. Open a browser in [http://localhost:3000](http://localhost:3000).
